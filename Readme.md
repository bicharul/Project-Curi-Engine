A robust, real-time web platform to combat motorcycle theft. Empowering communities to report stolen bikes and instantly check potential stolen vehicles.

## Features

- **Real-time Search**: Lightning-fast search using Elasticsearch
- **Stolen Bike Reporting**: Secure form for reporting stolen motorcycles
- **Community Alerts**: Notify the community about stolen bikes
- **Sighting Reports**: Crowdsourced bike sighting reports
- **Admin Dashboard**: Comprehensive admin interface for data management
- **RESTful API**: Complete API for integration and mobile apps

## Tech Stack

### Frontend
- React.js 18
- Tailwind CSS
- Axios for API calls
- React Router for navigation

### Backend
- Django & Django REST Framework
- PostgreSQL for data storage
- Elasticsearch for real-time search
- JWT Authentication

### Deployment
- Docker & Docker Compose
- Nginx for reverse proxy
